About Me

Saya adalah seorang web programmer, ilmuwan data, dan social media enthusiast. Dengan kecintaan terhadap teknologi, saya berkomitmen untuk terus belajar dan berkembang sehingga saya bisa menghasilkan karya dan inovasi bukan hanya untuk saya pribadi, namun juga untuk orang lain.

Dengan latar belakang saya, saya yakin dapat mengembangkan solusi kreatif serta efisien untuk tantangan teknis. Sebagai seorang ilmuwan data, saya mengeksplorasi dan menganalisis data untuk mendapatkan wawasan yang bernilai. Dan sebagai seorang penggemar media sosial, saya selalu bersemangat untuk terhubung dan berbagi pengetahuan yang saya miliki.
